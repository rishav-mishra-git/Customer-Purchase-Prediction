# Customer Purchase Prediction using Machine Learning

## Description
This project uses Machine Learning to predict whether a customer will purchase a product based on their age, gender, and estimated salary.

## Technologies Used
- Python
- NumPy
- Pandas
- scikit-learn

## Machine Learning Model
- Logistic Regression

## Workflow
1. Load and analyze customer data
2. Encode categorical variables
3. Split data into training and testing sets
4. Train a Logistic Regression model
5. Evaluate model performance using accuracy and confusion matrix

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the project:
   python model.py

## Result
The model predicts customer purchase behavior and provides accuracy and confusion matrix as evaluation metrics.

## Use Case
This project can help businesses understand customer behavior and improve marketing strategies.
